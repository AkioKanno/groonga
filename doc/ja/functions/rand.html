3.6.5. rand¶

3.6.5.1. 名前¶

rand - 乱数を生成する

3.6.5.2. 書式¶

rand([max])

3.6.5.3. 説明¶

groonga組込関数の一つであるrandについて説明します。組込関数は、script形式のgrn式中で呼び出すことができます。

rand() 関数は 0 から max の間の疑似乱数整数を返します。

3.6.5.4. 引数¶

max

返値の最大値を指定します。省略した場合は RAND_MAX が指定されたものとみなされます。

3.6.5.5. 返値¶

0 と max の間の数を表すInt32型の値を返します。

3.6.5.6. 例¶

rand(10)
3
